## 프로젝트 컨셉
![image](https://github.com/zeze1004/wedding-jira-backend/assets/44468282/4786e176-8b5d-4b00-9f7b-a1af37bbcf3f)
- JIRA처럼 할 일 카드를 칸반보드 식으로 진행 상황을 표현합니다.
